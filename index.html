Titulos e subtitulos

voltar

Os elementos < h1 > até o < h6 > em HTML são usados para definir os cabeçalhos ou títulos de uma página da web. Aqui está um resumo de cada um deles:

< h1 >: É o cabeçalho de nível mais alto e geralmente é usado para o título principal da página. Ele é o mais significativo em termos de hierarquia.

< h2 >: Um nível abaixo do < h1 >, usado para títulos de seção importantes. Normalmente, são subcategorias do título principal.

< h3 >: Usado para subseções dentro das seções < h2 >. São úteis para quebrar conteúdo em partes menores e mais gerenciáveis.

< h4 >, < h5 >, < h6 >: São cabeçalhos de níveis mais baixos, usados para títulos menos importantes e para subseções mais detalhadas dentro das seções anteriores. Geralmente, são usados quando a hierarquia precisa ser mais granular.

Esses elementos < h1 > a < h6 > não apenas fornecem estrutura e organização ao conteúdo da página, mas também são importantes para a acessibilidade da web e para otimização de mecanismos de busca, ajudando a destacar os principais tópicos e palavras-chave.

Exemplos:

H1 H2 H3 H4 H5 H6

Listas ordenadas e não ordenadas

voltar

Listar item < li >

List Item é usado para definir cada item em uma lista (< ol > ou < ul >). Cada < li > deve ser colocado dentro de um ol > ou < ul >< /ul >.

Lista ordenadas < ol >

É usado para criar uma lista ordenada, onde cada item é precedido por um número ou outro tipo de marcador de ordem

Exemplo:

  1. Objeto 1
  2. Objeto 2
  3. Objeto 3
  4. Objeto 4

Lista não ordenadas < ul >

É usado para criar uma lista não ordenada, onde cada item é precedido por um marcador de ponto, quadrado, círculo ou outro tipo de marcador não numérico.

Exemplo:

  • Objeto 1
  • Objeto 2
  • Objeto 3
  • Objeto 4
